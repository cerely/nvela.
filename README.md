# nvela. [Dashboard]

The Smart Lab Web Portal is part of the Smart Lab Automation System.  
It provides a central interface for managing lab activities such as attendance tracking, displaying notices, monitoring lab devices, and automating systems like fans and lights.

---

## Features

### Attendance Viewer
- Fetches and displays attendance data directly from the database.  
- Integrated with the QR-Based Attendance System for automated entry.  
- [QR Attendance System Repository](your-link-here)

---

### Notices
- Post and manage lab notices based on priority levels (High, Medium, Low).  
- Automatically highlights urgent notices on the main dashboard screen.

---

### Schedule
- Create and manage lab schedules for subjects, experiments, or sessions.  
- Supports daily and weekly views for better organization.

---

### Document Viewer
- Upload and view multiple documents within the same browser tab.  
- Supports PDF, DOCX, and other common file formats.

---

### Screen Saver Mode
- Automatically activates during idle periods to display lab information or announcements attractively.

---

## Tech Stack

**Client:** HTML, CSS, PHP, JavaScript  
**Server:** MySQL, Apache
